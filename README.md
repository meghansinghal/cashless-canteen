****Cashless Canteen – DBMS Mini Project****

**Team Members:** Meghan Singhal, Mungara Shreya <br>
**Tech Stack:** Streamlit (Python), MySQL, MySQL Connector, SQL Views + Procedures + Triggers

A complete cashless canteen management system where students can: <br>
✔ Browse menu <br>
✔ Add items to cart <br>
✔ Apply coupons <br>
✔ Top-up wallet <br>
✔ Make online payments <br>
✔ View order history <br>
✔ Admins can update menu items

This project demonstrates complete DBMS integration, including:
- Strong database schema
- Triggers
- Stored procedures
- Views
- Constraints
- Transactional order creation
